Internet

The internet is a global network of computer networks that allows us to communicate information. It is literally physically connected through underwater cables around the planet.

Web

Known as the World Wide Web, it is a service of web pages and information that run within the internet.

How are the Internet and Web different?

The internet is the infrastructure that connects everything (like a highway). The web, on the other hand, is like the cars that travel on that highway (what you can actually see).

ASCII and Unicode

ASCII is a dictionary that translates the numbers that the computer can process into an alphabet that is readable for us. And Unicode is the comprehensive dictionary readable by all the languages ​​of the world (ASCII only has 128 characters for the English language, while Unicode has many more).

What is UTF-8

It is a system of characters just like Unicode, but unlike other sistems, It uses diferent amount of bytes and shearable with diferent countries of the world.

URL

Uniform Resource Locator. The URL is the address through which the information passes.

Domain Name

It is the name assigned to an IP or internet address. Its function is to prevent us from having to memorize complex numbers in order to access a particular website.

HTTP

Stands for HyperText Transfer Protocol, and and it is the set of rules that allows us to exchange information between our browser and our server.

HTTPS

Unlike HTTP, HTTPS provides us with greater security. Almost all websites today use this protocol, which prevents information leaks.

IP Address

IP stands for Internet protocol, and it is the numerical label that allows us to identify our website or our devices.

DNS

It stands for Domain Name system, and it is the service that translates the website names we type into the numerical addresses that computers understand.

W3C Recommendations

OThese are the official standards of the World Wide Web Consortium that tell us how web technologies should work.

XML, XHTML, HTML, HTML5

XML: (eXtensible Markup Language) It is a language that structures and transmits data. This allows us to help store data in an organized manner.

XHTML: (eXtensible HyperText Markup Language). It is a stricter version of XML for cleaning and structuring our data.

HTML: (HyperText Markup Language). It is the basic language of the web and creates the structure of web pages.

HTML5: It is the latest version of HTML and is the one currently used to create web pages.
