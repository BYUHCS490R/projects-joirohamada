The form tag has an "action" attribute. What does it do? The action attribute indicates where we are going to send the data we are entering, or, in other words, it tells us where all that information is going.

The form tag has a "method" attribute. What does it do? It specifies how the form data will be sent.

What is the difference between a "POST" and a "GET"? post = something changes. (on the serve in database) get = search/filter/sort


What are different ways to style your form? We can style our forms with CSS chnging colors, fonts, borders etc, and by using pseudo-classes like focus. We can also use Flex to organize our elements.

How does flex help for both Desktop and Mobile phones? Flex is very useful because it allows elements to be organized in a row on desktop and in a column on mobile mking the form more organized for different screens.
