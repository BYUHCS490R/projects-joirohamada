Engagement questions

Why do we need a viewport meta tag? The viewport meta tag is a very important HTML tag because it specifies how the page will appear depending on the type of device we use to see the web page. Without this tag, our page would be either too large or too small, forcing us to scroll horizontally to see the entire page on our device.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? The tag that was placed at the end has the priority.

How do you change the CSS for a web page displayed on a phone? We use the @media option and we can also adjust the pixel size we want on our page so that our page adjusts according to the device we want to open it on.
