Engagement questions

Why is the viewport meta tag important? The meta tag is important because it allows us to adjust the page to different devices based on their screen sizes. For instance, a mobile phone screen has different dimensions than a computer or television screen. This allows the page to adapt to the needs of the screens.

How can you adjust the CSS for printing? By using the @media print function as one of our classes

When would you want to adjust the CSS for printing? When we want to print something, sometimes all we need is the page content, but maybe we don't need the nav tag, so the @media print function is very useful.
