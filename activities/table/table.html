Why don't we use the border attribute? We don't use the border option because we can modify our table in CSS. We can handle the design of the table and its borders using CSS instead of HTML.

What is the easiest way to color every other row a different color? We can use the option tr:nth-of-type(odd) or tr:nth-of-type(even). This option selects the rows or columns we want to configure when we are designing our table.
