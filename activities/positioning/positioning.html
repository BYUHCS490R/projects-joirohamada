What is normal flow in HTML? Flow is the established norm in which the elements of a web page are arranged naturally.

What does float do? Why do you need to clear the float? float is a tool that allows us to make an element move to the left or right, and have text or other elements surround it. The clear command allows us to reorder the image we are using, causing the element we are editing to no longer surround the original element or image.

What do absolute, fixed, and relative do? The `relative` option moves an element from its normal position, but it still occupies its original space. The absolute command places the element in a specific position within its container. The fixed command is very useful when we want to fix an element in a specific position on the screen. No matter how much we scroll down the page, the fixed element will remain visible.
