Beginning CSS

CSS: Casscading Style Sheets.

1. Explain the difference between inline, embedded, and external CSS. INLINE is within one line. Embedded is under the head section, only modfying one page. The external CSS lets us modify several pages.

2. Explain why you should only use external CSS. When we create a web page as a team, usually people working on the design of the page (usually graphic designers) work separately from the HTML. If it is all together in one project, sometimes it can mess the program. Since we create several pages, when we have an external css, we can just link our css program, we can link the same project to several pages.

3. What is a selector used for? To select which tag we want to modify in our page, and it is located outside of the brackets in the CSS program. (If you want to modify the paragraph tag, you specify by using the letter P).
